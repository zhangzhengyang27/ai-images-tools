## 一、整体规划

### 1.1 开发阶段划分

本项目分为 **4 个阶段**，共 14 个迭代任务，预计开发周期 5~7 周。

| 阶段 | 周期 | 目标 | 交付物 |
|---|---|---|---|
| **Phase 1 · 骨架搭建** | 第 1~2 周 | 项目初始化、目录结构、基础 UI 布局、主进程框架 | 可运行的空壳 Electron 应用，带 UI 布局 |
| **Phase 2 · 核心功能** | 第 3~4 周 | 图片上传、压缩引擎、对比视图、导出保存 | P0 功能全部可用 |
| **Phase 3 · 完善交付** | 第 5~6 周 | 批量处理、尺寸调整、UI 打磨、打包发布 | exe/dmg 可执行文件 |
| **Phase 4 · 历史记录** | 第 6~7 周 | 压缩历史记录、持久化存储、历史面板 UI | 历史记录功能完整可用 |

### 任务卡 3：Sharp 图片压缩引擎
